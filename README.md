# AvDrom

## 📌 Описание проекта
AvDrom — это веб-приложение для просмотра и покупки автомобилей.  
Пользователи могут зарегистрироваться, войти в аккаунт и просматривать доступные автомобили.  
Сайт предоставляет простой и удобный интерфейс для поиска машин.  
Проект реализован с использованием Node.js и базы данных MySQL.

---

## 👥 Целевая аудитория
- Люди, которые хотят купить автомобиль  
- Пользователи, ищущие удобный каталог машин  
- Начинающие автолюбители  

---

## ⚙️ Основной функционал
- Регистрация пользователей  
- Авторизация (вход в аккаунт)  
- Просмотр каталога автомобилей  
- Просмотр информации о машине  
- Переход между страницами сайта  

---

## 🗂 Структура приложения

1. Главная (/) – публичная  
2. Аутентификация (/login, /register) – вход и регистрация  
3. Каталог автомобилей (/cars) – список машин  
4. Страница автомобиля (/cars/:id) – подробная информация  
5. Личный кабинет (/profile) – приватный раздел  
6. Избранное (/favorites) – сохранённые авто  
7. Поиск (/search) – поиск автомобилей  
8. О нас (/about) – информация о сайте  

---

## 🧰 Технологический стек

### Frontend:
- HTML — структура страниц  
- CSS — стилизация интерфейса  
- JavaScript — логика на клиенте  

📌 Почему:  
Простые технологии, подходят для обучения и быстрого старта.

---

### Backend:
- Node.js — серверная часть  
- Express — обработка запросов  

📌 Почему:  
Позволяет быстро создавать API и работать с формами.

---

### База данных:
- MySQL  

📌 Почему:  
Надёжная и популярная СУБД для хранения пользователей и данных автомобилей.

## 🖥 Визуальное описание интерфейса

### Страница: Главная (/)
Тип доступа: публичный  
Блоки:
1. Шапка — логотип, меню (Главная, О нас, Войти)  
2. Баннер — приветствие и кнопка  
3. Популярные автомобили — карточки  
4. Футер  

Динамика: карточки загружаются с сервера  

---

### Страница: Логин (/login)
Тип доступа: публичный  
Блоки:
1. Форма входа (логин, пароль)  
2. Кнопка входа  
3. Ссылка на регистрацию  

---

### Страница: Регистрация (/register)
Тип доступа: публичный  
Блоки:
1. Форма регистрации  
2. Поля: логин, пароль  
3. Кнопка регистрации  

---

### Страница: Каталог (/cars)
Тип доступа: публичный  
Блоки:
1. Список автомобилей  
2. Карточки с фото, ценой и кнопкой  

Динамика: данные из базы данных  

---

### Страница: Автомобиль (/cars/:id)
Тип доступа: публичный  
Блоки:
1. Фото автомобиля  
2. Описание  
3. Цена  

---

### Страница: Профиль (/profile)
Тип доступа: приватный  
Блоки:
1. Данные пользователя  
2. Кнопка выхода  

---

### Страница: Избранное (/favorites)
Тип доступа: приватный  
Блоки:
1. Список сохранённых авто  

---

### Страница: О нас (/about)
Тип доступа: публичный  
Блоки:
1. Информация о компании  
2. Текст  

---

- price

## 🌐 API

- POST /register — регистрация пользователя  
- POST /login — вход в аккаунт  
- GET /cars — получить список автомобилей  
- GET /cars/:id — получить данные об автомобиле  

---

## 🚀 Запуск проекта

1. Установить зависимости:
npm install

2. Запустить сервер:
node server.js

3. Открыть в браузере:
http://localhost:3000
